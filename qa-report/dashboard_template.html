⚡ ChargeRoute — EV Route & Charging Planner QA Dashboard

1,000 designed test runs against the app's real routing/validation logic (calculateRoute + form validation), network layer replaced by a deterministic synthetic world model. Generated 2026-07-18.

Methodology note: this suite drives the app's business-logic layer directly (not the Android UI/emulator) — see ev_test_report.md for the full rationale. The network/API layer (Google Directions, Places, OpenChargeMap, SQLite charger cache) is replaced by a seeded synthetic world model (50 real named locations across India, a charger-density grid) so results are exactly reproducible. Response times are logic-layer computation time (sub-millisecond to ~14ms), not real device/network latency — there is no network call in this harness, so the response-time charts measure algorithm cost, not user-perceived speed.
